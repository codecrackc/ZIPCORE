# 🗜️ ZipCore

**ZipCore** es un compresor/descompresor de archivos profesional para Windows, completamente **GRATUITO** y sin límites de uso. Diseñado para ser rápido, ligero y fácil de usar.

## ✨ Características Principales

- ✅ **Completamente Gratuito** - Sin licencias, sin pagos, sin límites
- ✅ **Múltiples Formatos** - Soporta ZIP, RAR y TAR
- ✅ **Ultra Rápido** - Procesamiento optimizado para máxima velocidad
- ✅ **Interfaz Sencilla** - Diseño intuitivo y fácil de usar
- ✅ **Sin Publicidad** - Experiencia limpia y profesional
- ✅ **Ligero** - Menos de 10 MB de tamaño
- ✅ **Actualizaciones** - Sistema de actualización integrado
- ✅ **Verificación MD5** - Comprueba la integridad de tus archivos
- ✅ **Compresión Multinivel** - Rápida, Normal o Máxima compresión
